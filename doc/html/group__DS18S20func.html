uint8_t DS18x20_Init ( TSDS18x20 *  pDS18x20,
volatile uint8_t *  DS18x20_PORT,
uint8_t  DS18x20_PIN 
)

Init function for DS18x20. The function prepares the 1-Wire bus and runs the sensors initialization procedure. In case of DS18B20, the init function doesn't set the value of the Configuration register. After the initialization is done it still has the default value. To change the resolution use DS18x20_WriteScratchpad() function or DS18x20_SetResolution() macro.

Parameters
[in,out]pDS18x20pointer to the structure that represent DS18S20
[in]DS18x20_PORTis a pointer to the MCU PORT used for the 1-Wire bus
[in]DS18x20_PINMCU PIN from DS18x20_PORT connected to the DS18S20 sensor
Returns
uint8_t. 0 for success init, 1 for failure.
